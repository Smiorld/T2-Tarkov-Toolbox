# T2 Tarkov Toolbox - 开发路线图

## 当前版本功能 (v1.0)

### 核心功能
T2 Tarkov Toolbox 目前提供两大核心功能模块：

#### 1. 屏幕滤镜 (Screen Filter)
- **功能描述**：通过Windows GDI gamma ramp控制显示器颜色
- **主要特性**：
  - 多预设管理（创建、编辑、删除、复制）
  - 快捷键切换预设
  - 多显示器支持
  - 实时参数调整：
    - 亮度 (Brightness)
    - 伽马 (Gamma)
    - 对比度 (Contrast)
    - RGB通道独立调整
  - 悬浮窗对冲设置（防止地图悬浮窗过曝）
  - 参数安全验证机制
  - 配置持久化存储

#### 2. 本地地图 (Local Map)
- **功能描述**：实时显示玩家在地图上的位置和朝向
- **主要特性**：
  - 多地图支持（工厂、海关、森林、海岸线等）
  - 多层级地图（地面、地下）
  - 实时位置追踪（通过解析游戏日志）
  - 悬浮窗小地图：
    - 可拖拽、调整大小
    - 透明度调整
    - 锁定/解锁模式
    - 滚轮缩放支持
    - 玩家居中模式
  - 校准系统：
    - 多点校准（3-10个点）
    - 多种算法支持（最小二乘法、加权最小二乘法、RANSAC）
    - 误差验证
  - 坐标系统转换（游戏坐标 ↔ 地图像素坐标）
  - 状态持久化（悬浮窗位置、缩放级别、透明度等）

#### 3. 全局设置 (Global Settings)
- **功能描述**：应用级别的通用设置
- **主要特性**：
  - 主题切换
  - 语言设置（预留）
  - 其他全局配置

### 技术架构
- **UI框架**：CustomTkinter (基于Tkinter)
- **图像处理**：PIL/Pillow
- **数值计算**：NumPy
- **系统API**：
  - Windows GDI (gamma ramp控制)
  - 文件系统监控 (Watchdog)
  - 全局快捷键 (keyboard库)

## 未来功能规划

### 📋 任务追踪器 (Quest Tracker) - 计划中

#### 功能概述
提供塔科夫任务进度追踪和物品清单管理功能。

#### 核心功能设计
1. **任务列表管理**
   - 显示所有可用任务
   - 按商人/阵营分组
   - 任务搜索和过滤
   - 任务状态标记（未接取、进行中、已完成）

2. **任务详情展示**
   - 任务目标清单
   - 所需物品列表
   - 任务奖励信息
   - 前置任务依赖关系
   - 地图位置标注

3. **进度追踪**
   - 任务进度实时记录
   - 物品收集进度
   - 击杀/生存/探索进度
   - 自动检测游戏日志更新进度（可选）

4. **物品管理**
   - 任务所需物品清单
   - 已收集物品标记
   - 物品获取途径提示
   - 藏匿物品提醒

5. **个性化功能**
   - 自定义任务优先级
   - 笔记和标签
   - 任务路线规划建议
   - 导入/导出进度数据

#### 技术实现考虑
- **数据源**：
  - 静态任务数据库（JSON/SQLite）
  - 官方Wiki API（如果可用）
  - 社区数据源

- **UI设计**：
  - 树形结构展示任务依赖
  - 可折叠的任务卡片
  - 进度条和统计图表
  - 搜索和快速过滤

- **数据持久化**：
  - 本地JSON配置文件
  - 云同步支持（可选）

#### 开发优先级
- 优先级：**中等**
- 预计工作量：2-3周
- 前置条件：
  - 完善现有两个模块
  - 收集完整的任务数据
  - 用户需求调研

#### 潜在挑战
- 任务数据的完整性和及时更新
- 游戏版本更新导致的任务变化
- 多语言支持
- 复杂的任务依赖关系可视化

## 开发原则

1. **专注核心功能**：优先完善现有的屏幕滤镜和本地地图功能
2. **模块化设计**：保持各模块独立，便于维护和扩展
3. **用户体验优先**：界面简洁、操作流畅、性能优秀
4. **稳定性第一**：充分测试，避免影响游戏体验
5. **社区驱动**：根据用户反馈调整开发优先级
